Integer

  1. model will be number or null
  2. only use with integers
  3. when model is null
    1. Subtract button will set model to min
    2. Add button will set model to max
  4. when model is less than min
    1. Subtract button will be disabled
    2. Add button will set model to min
  5. when model is greater than max
    1. Add button will be disabled
    2. Subtract button will set model to max

Inputs

  • disabled can be an attribute, when specified it will be boolean. default is false
  • required will be boolean. default is false
  • showNull will be boolean. default is false
  • min will be number. default is 0
  • max will be number. default is 9
  • step will be number. default is 1
