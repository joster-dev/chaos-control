Small Number

  1. control value will be number or null
  2. only use with integers
  3. when control value is null
    1. Minus button will set control value to min
    2. Add button will set control value to max
  4. when control value is less than min
    1. Minus button will be disabled
    2. Add button will set control value to min
  5. when control value is greater than max
    1. Minus button will set control value to max
    2. Add button will be disabled
default: min is 0 & max is 9 & step is 1
