Form Control Documentation

github v0.0.8

About

  • customizable predictable control over form elements
    • error messages when invalid model values are provided
  • keyboard friendly
    • tab-index-able
    • conforms to standard layout
    • handles up and down keys for incrementing

Getting Started

  1. in your angular project run
    • npm install @joster-dev/form-control @joster-dev/icon --save
  2. in the module that utilizes these controls
    1. import the dependency
      • {{"import { FormControlModule } from '@joster-dev/form-control'"}}
    2. update your dependencies
      • imports: [..., FormControlModule]
