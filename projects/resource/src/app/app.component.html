Form Control Resource

github v0.0.4

About

  • styled controls for forms
    • helpful error messages when invalid input values are provided
  • accessible
    • passes WCAG AA standards
  • keyboard friendly
    • tab-index-able
    • conforms to standard layout
    • handles up and down keys for incrementing
  • touchscreen friendly
    • handles long-press for incrementing
  • visual indication for control statuses
    • error text when invalid
  • print friendly
    • always see selected value
    • removal of borders for a clean print

Getting Started

  1. in your angular project run
    • npm install @joster-dev/form-control --save
    • npm install @joster-dev/icon --save
  2. in the module that utilizes these controls add
    • {{"import { FormControlModule } from '@joster-dev/form-control'"}}
  3. in the module update your imports
    • imports: [..., FormControlModule]
